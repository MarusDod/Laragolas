# LARAGOLAS

GUI do laravel para Linux.

![Alt text](https://raw.githubusercontent.com/MarusDod/Laragolas/master/laragolas.png "laragolas")

## Requisitos
*   Ter uma Debian based distro
*   Systemd
*   Usar GNOME como DE

## Dependências
*   libgtk-3-dev
*   mysql
*   mysql-workbench
*   make
*   gcc
*   composer
*   php
*   laravel

## Instalação

```bash
   git clone https://github.com/MarusDod/Laragolas
   cd Laragolas
   make release
   make install
```

## Correr
```
   laragolas.out
```
