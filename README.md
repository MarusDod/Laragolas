# LARAGOLAS

GUI do laravel para Linux.

## Requisitos
*   Ter uma Debian based distro
*   Systemd
*   Usar GNOME como DE

## Dependências
*   libgtk-3-dev
*   mysql
*   mysql-workbench
*   make
*   gcc
*   composer
*   php
*   laravel

## Instalação
*   git clone https://github.com/MarusDod/Laragolas
*   cd Laragolas
*   make

## Correr
*   ./laragolas.out
